# CyberGuard - Your Digital Defense

A responsive landing page developed as a university extension project aimed at educating users about cybersecurity and online safety. The platform provides information on common cyber threats, phishing attacks, social engineering scams, password security, and best practices for protecting personal data.

The main goal of CyberGuard is to raise digital awareness and help users identify potential online threats, avoid scams, and browse the internet more safely and confidently.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Cybersecurity awareness content
* Scam and phishing prevention tips
* Responsive design for desktop and mobile devices
* User-friendly interface
